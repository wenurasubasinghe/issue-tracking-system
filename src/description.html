GitHub Issue Time Tracker

Welcome to the GitHub Issue Time Tracker extension! This powerful tool is designed to help you stay organized and accountable for your time spent working on GitHub issues and pull requests.

Features

  • Start and stop a timer directly from the GitHub issue or pull request page
  • Automatically associate the timer with the current issue or pull request
  • Provide a task description for each timed activity
  • View the total time spent on each issue or pull request
  • Seamless integration with your GitHub workflow

How to Use

  1. Navigate to a GitHub issue or pull request page
  2. Click the "Start Timer" button in the sidebar
  3. Select the activity type (e.g., coding, reviewing, etc.)
  4. Provide a brief description of the task
  5. Work on the issue or pull request
  6. When you're done, click the "Stop Timer" button to log the time spent

The GitHub Issue Time Tracker extension is designed to help you stay organized and accountable for your time spent on GitHub projects. With its seamless integration and user-friendly interface, you can easily track your progress and ensure you're making the most of your time.
